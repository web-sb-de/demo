Häufige Fragen

Welche Vorteile hat eine Mitgliedschaft?

Eine Mitgliedschaft bietet Ihnen Zugang zu all unseren Veranstaltungen, Workshops und Projekten. Sie erhalten den monatlichen Gartenratgeber, profitieren von der Gemeinschaft und können aktiv an der Förderung nachhaltiger Gartenpraktiken teilnehmen.

Kann ich mir den Verein erstmal ansehen?

Ja, natürlich! Kommen Sie einfach bei einer unserer Veranstaltungen vorbei und lernen Sie uns kennen. Eine Übersicht der Termine finden Sie auf unserer Terminübersicht.

Ist Mithilfe Pflicht?

Nein, Mithilfe ist freiwillig. Sie können so viel oder so wenig beitragen, wie Sie möchten. Wir freuen uns über jede Unterstützung, aber es gibt keine Verpflichtung.

Wie hoch ist der Mitgliedsbeitrag?

Der jährliche Mitgliedsbeitrag beträgt 25 Euro. Für Familien gibt es ermäßigte Tarife. Kontaktieren Sie uns für weitere Details.

Wie kann ich Mitglied werden?

Füllen Sie den Aufnahmeantrag aus und senden Sie ihn an die Vorstandschaft. Sie finden das Formular weiter unten auf dieser Seite oder kontaktieren Sie uns direkt.
